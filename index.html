Concepts of the Web and HTML

The World Wide Web

The world wide web is a make up of HTML documents. Each computer browser comunicates with servers through a protocol called HTTP or Hypertext Transfer Protocol. HTTP is used by computers to communicate across the world wide web. This video breaks down the process of communication between browsers and servers in an easy to understand fashion.


HTML

HTML means Hypertext Markup language. HTML is made up of text content ("what you see"), mark up ("what it looks like"), references to other documents (such as images or videos), and links to other pages. Click here to see a reference to HTML.

Tags

HTML markup is made up of tags. Tags are like keywords that tell a web page how to show content. HTML documents are made up of HTML elements. An element is everything from the opening tag to the closing tag. Here is a good example of tags and elements.

Inline and Block Elements

An Inline element is an element that takes up only the space needed. A Block element creates a sort of invisible box around the text that is being tagged. An example of a block element is the P tag, or paragraph tag, which creates a space before and after the text that is written.
